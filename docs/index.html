Install as trasgo

npm install -g trasgo
trasgo --help

npm ci
node scripts/trasgo-launch.cjs --help
.\\trasgo.cmd --help   # Windows
./bin/trasgo --help    # macOS/Linux

cargo build --manifest-path rust/trasgo/Cargo.toml --release
rust/trasgo/target/release/trasgo.exe --help   # Windows
rust/trasgo/target/release/trasgo --help       # macOS/Linux

State lives in ./.trasgo-runtime by default. Set TRASGO_HOME to relocate sessions, packs, and traces.

The npm package, installed command, and Rust crate all use trasgo.

Core workflow

trasgo init "portfolio runtime"
trasgo pack --out .trasgo-runtime/packs/portfolio.json
trasgo boot --from .trasgo-runtime/packs/portfolio.json
trasgo send "What changed for K and why?"
trasgo bench lmstudio
trasgo calibrate

One verb, one shell, one runtime surface.
